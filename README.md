# Medical Linear Regression - Glucose Prediction

This project applies Linear Regression to predict blood glucose levels
using basic clinical features such as age, BMI, and blood pressure.

⚠️ Dataset is synthetic and created for educational purposes only.

## Features
- Medical data preprocessing
- Multi-variable Linear Regression
- Model evaluation (MSE, R²)
- Interpretability via coefficients
